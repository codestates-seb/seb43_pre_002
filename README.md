## Git
### Branch
* main : 서비스 운영 브랜치
* dev : 개발중인 브랜치
* feat/기능명 : 기능 개발 브랜치

### Commit Message
|Message	|설명|
|:---:|:---:|
|Feat | 새로운 기능 추가|
|Fix | 버그 수정|
|Docs | 문서 수정|
|style | 코드 포맷팅, 세미콜론(;) 누락, 코드 변경이 없는 경우|
|Refactor | 코드 리팩터링|
|Test | 테스트 코드, 리팩터링 테스트 코드 추가|
|Chore | 빌드 업무 수정, 패키지 매니저 수정|
|Rename | 파일 혹은 폴더명을 수정하거나 경로 이동|
|Remove | 파일을 삭제|
